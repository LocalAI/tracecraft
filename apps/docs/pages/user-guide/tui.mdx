# Terminal UI

TraceCraft includes an interactive terminal UI for exploring traces.

## Installation

```bash
pip install "tracecraft[tui]"
```

## Quick Start

```bash
# View traces from JSONL file
tracecraft tui traces.jsonl

# View traces from SQLite database
tracecraft tui traces.db

# View traces from directory
tracecraft tui traces/
```

## Features

- Browse trace hierarchy
- Filter by name, status, duration
- Search spans
- View detailed span information
- Export individual traces

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `↑↓` | Navigate traces |
| `←→` | Navigate details |
| `Enter` | Expand/collapse |
| `Tab` | Switch views |
| `/` | Search |
| `f` | Filter |
| `e` | Export trace |
| `r` | Refresh |
| `q` | Quit |
| `?` | Help |

## Views

### Trace Table

List of all traces with summary information.

### Waterfall View

Visual timeline of span execution.

### Detail View

Complete span information including attributes, inputs, outputs.

## Filtering

Press `f` to open filter dialog:

```
Status: [All | Success | Error]
Duration: [Min] - [Max] ms
Name: [Pattern]
```

## Exporting

Press `e` to export current trace to:

- JSONL file
- HTML report
- Clipboard (JSON)
