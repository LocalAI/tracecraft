# Exporters API

Export backends for TraceCraft.

## BaseExporter

Base class for all exporters.

```python
from tracecraft.exporters import BaseExporter
from tracecraft.core.models import AgentRun

class MyExporter(BaseExporter):
    def export(self, run: AgentRun) -> None:
        # Export logic
        pass
```

## ConsoleExporter

Rich terminal output.

```python
from tracecraft.exporters import ConsoleExporter

exporter = ConsoleExporter(verbose=True)
```

## JSONLExporter

Write traces to JSONL files.

```python
from tracecraft.exporters import JSONLExporter

exporter = JSONLExporter(
    filepath="traces.jsonl",
    append=True,
)
```

## OTLPExporter

Export to OTLP-compatible backends.

```python
from tracecraft.exporters import OTLPExporter

exporter = OTLPExporter(
    endpoint="http://localhost:4317",
    insecure=True,
    headers={"Authorization": "Bearer token"},
)
```

## MLflowExporter

Export to MLflow.

```python
from tracecraft.exporters.mlflow import MLflowExporter

exporter = MLflowExporter(
    tracking_uri="http://localhost:5000",
    experiment_name="my-agent",
)
```

## HTMLExporter

Generate HTML reports.

```python
from tracecraft.exporters import HTMLExporter

exporter = HTMLExporter(
    output_dir="reports/",
    include_source=True,
)
```
