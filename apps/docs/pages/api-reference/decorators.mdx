# Decorators API

Instrumentation decorators for TraceCraft.

## trace_agent

Decorator for agent orchestration functions.

```python
from tracecraft import trace_agent

@trace_agent(
    name="my_agent",
    exclude_inputs=["secret"],
    capture_inputs=True,
    runtime=None,
)
async def my_agent(query: str) -> str:
    pass
```

## trace_tool

Decorator for tool/utility functions.

```python
from tracecraft import trace_tool

@trace_tool(name="math_operation")
def math_operation(a: float, b: float) -> float:
    return a + b
```

## trace_llm

Decorator for LLM API calls.

```python
from tracecraft import trace_llm

@trace_llm(
    name="chat",
    model="gpt-4",
    provider="openai",
)
async def chat(prompt: str) -> str:
    pass
```

## trace_retrieval

Decorator for retrieval operations.

```python
from tracecraft import trace_retrieval

@trace_retrieval(name="search")
async def search(query: str) -> list[str]:
    pass
```

## step context manager

Context manager for manual instrumentation.

```python
from tracecraft import step
from tracecraft.core.models import StepType

with step("processing", type=StepType.WORKFLOW) as s:
    s.inputs["data"] = data
    result = process(data)
    s.outputs["result"] = result
```
